
Project Overview
Hobbyist is a web based application that allows users to create personal curriculums, track progress and manage their learning

User flows:
Sign up
Login
Creating different types of 'lessons' [video, quiz]
Removing lessons
Marking lessons as complete
Tracking progress

## Technologies:

This project is built with:

- Supabase
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
